T.13：

给定一条空间封闭折线 $A_1A_2...A_n,$ 某个平面与这条折线的每条线段都相交，设与 $A_1A_2$ 的交点是 $B_1,$ 与 $A_2A_3$ 的交点是 $B_2,...,$ 与 $A_nA_1$ 的交点是 $B_n,$

请证明： $\frac{A_1B_1}{B_1A_2}\cdot\frac{A_2B_2}{B_2A_3}\cdot...\frac{A_nB_n}{B_nA_1}=1。$
